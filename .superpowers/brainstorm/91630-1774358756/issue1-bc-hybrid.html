🧠 Adaptive Logic — "Tùy chỉnh thêm" hiện fields nào?

Field 🌱 Beginner 💪 Intermediate 🏆 Advanced
Luôn hiện (Quick Start — 3 core fields):
training_goal ✅ ✅ ✅
experience_level ✅ ✅ ✅
days_per_week ✅ ✅ ✅
Expand section — "Tùy chỉnh thêm":
session_duration_min ✅ hiện ✅ hiện ✅ hiện
available_equipment ✅ hiện ✅ hiện ✅ hiện
injury_restrictions ✅ hiện ✅ hiện ✅ hiện
cardio_sessions + type ✅ hiện ✅ hiện ✅ hiện
Advanced sub-section (chỉ hiện nếu Intermediate/Advanced):
periodization_model ẩn ✅ hiện ✅ hiện
plan_cycle_weeks ẩn ✅ hiện ✅ hiện
priority_muscles ẩn ✅ hiện ✅ hiện
known_1rm ẩn ẩn ✅ hiện
avg_sleep_hours ẩn ẩn ✅ hiện
Tổng kết fields theo level:
🌱 Beginner: 3 core + expand → tối đa 7 fields (equipment, injuries, cardio, duration)
💪 Intermediate: 3 core + expand → tối đa 10 fields (+ periodization, cycle, muscles)
🏆 Advanced: 3 core + expand → tối đa 12 fields (+ 1RM, sleep)

🗺️ User Journey — 3 personas

🌱 Mai — Beginner
1. Chọn: Giảm mỡ → Mới → 3 ngày
2. Thấy AI preview → "OK hợp lý"
3. Không bấm "Tùy chỉnh"
4. Bấm "Tạo plan" → 20s xong
5. Bắt đầu tập ngay!
✅ Zero friction
💪 Tuấn — Intermediate
1. Chọn: Tăng cơ → Trung bình → 4 ngày
2. Bấm "Tùy chỉnh thêm"
3. Chỉnh: 60'/buổi, chỉ Dumbbell + Bodyweight
4. Thêm: 1 buổi cardio HIIT
5. Bấm "Tạo plan" → 45s xong
✅ Chỉ chỉnh cái cần
🏆 Hùng — Advanced
1. Chọn: Sức mạnh → Nâng cao → 5 ngày
2. Bấm "Tùy chỉnh thêm"
3. Chỉnh equipment, cardio
4. Thấy Advanced section → chọn Undulating, 6 tuần
5. Nhập 1RM: Squat 180, Bench 120
6. Bấm "Tạo plan" → 90s xong
✅ Full control nếu muốn
