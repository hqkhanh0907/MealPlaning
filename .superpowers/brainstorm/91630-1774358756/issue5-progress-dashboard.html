Charts nào glanceable nhất? Insight-first vs Data-first approach

📌 Spec hiện tại — Scroll dọc dài

Spec hiện tại (stack dọc):
1. Cycle Progress Bar
2. Volume Trend Line Chart (8 tuần)
3. Weight Trend Line Chart (daily + moving avg)
4. Stats Cards row (adherence, 1RM, sessions)
⚠️ 2 line charts liên tiếp → visual monotony
⚠️ Scroll 3-4 screens để xem hết
⚠️ Chỉ hiện raw data — user tự phải đọc trend
⚠️ Mobile screen nhỏ → chart detail bị mất
Fitness app benchmarks:
Strong App: Hero number + sparkline mini
Hevy: Swipeable metric cards
Apple Fitness: Activity rings + trends
MyFitnessPal: Weekly summary banner

Common pattern:
✅ 1 hero metric lớn → glanceable 2 giây
✅ Sparklines thay vì full charts
✅ AI/text insights thay vì raw numbers
✅ Swipe horizontal thay vì scroll dọc
