Flow A — Workout Logging: Luồng chính của module Fitness. Người dùng bắt đầu buổi tập, nhập thông số từng set (weight/reps/RPE), hệ thống lưu draft tự động, sau đó hoàn thành buổi tập với atomic transaction và phát hiện PR (Personal Record).
