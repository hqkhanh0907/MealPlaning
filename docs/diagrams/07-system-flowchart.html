⇄ Tích Hợp Liên Module

🔗 Fitness → Nutrition

  • Calorie burn từ workout → điều chỉnh daily calorie targets
  • Activity multiplier → tính toán BMR / TDEE chính xác
  • Thay đổi cân nặng → cập nhật macro targets
  • Training intensity → đề xuất dinh dưỡng phục hồi

📊 Fitness → Dashboard

  • Trạng thái workout hôm nay (đã tập / chưa tập)
  • Quick action buttons (bắt đầu tập)
  • Streak display (chuỗi tập liên tiếp)
  • Weight mini widget (cân nặng hôm nay)

👤 Fitness → Health Profile

  • Cân nặng user → ước lượng calories cardio (MET × kg)
  • Tuổi user → khuyến nghị phục hồi phù hợp
  • BMI tracking → điều chỉnh training recommendations
