Design Notes — Meals

  • Pure grayscale: #0A0A0A → #FAFAFA spectrum
  • Macro tags dùng text label (Pro, Fat, Carb, Fiber) thay vì màu
  • Progress bar đen trên xám — không cần màu để truyền đạt %
  • CTA "Lên kế hoạch" = filled black, secondary = outlined
  • Date selector: active = inverted (đen nền trắng chữ)
  • Empty slot: dashed border + gray, dễ nhận biết chưa điền

Design Notes — Nutrition

  • Donut chart dùng opacity (100% / 50% / 20%) thay vì màu sắc
  • Fiber dùng bordered swatch (outline) để phân biệt nhóm khác
  • Mỗi macro có text label rõ ràng: "Protein", "Fat", "Carb", "Fiber"
  • Progress bars đen trên xám — đồng nhất toàn hệ thống
  • BMR/TDEE/Target = 3 cards ngang — editorial typography style
  • Recommendation card = neutral gray, không dùng semantic color
